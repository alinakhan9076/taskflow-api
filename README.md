# TaskFlow Backend

TaskFlow Backend is built using Node.js, Express.js, MongoDB Atlas, and JWT Authentication. It provides secure REST APIs for user authentication and task management.

## Features

- User Registration
- User Login
- Password Hashing using bcrypt
- JWT Authentication
- Protected Routes
- Create Task
- Read Tasks
- Update Task
- Delete Task
- MongoDB Atlas Integration

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcrypt

## API Base URL

https://taskflow-api-production-26f6.up.railway.app
